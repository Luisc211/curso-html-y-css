Etiqueta <a>

La etiqueta <a> nos permite crear enlaces o links, significa anchor o ancla, y se utiliza para crear hipervinculos, es decir, conectar una página web con otra, o incluso con una parte específica de la misma página, y su estructura es la siguiente: <a href="URL"> Texto del enlace </a>.

Ejemplo

Abrir enlaces en otra pestaña

Para abrir un enlace en otra pestaña del navegador agregamos el atributo target dentro de la etiqueta <a>
