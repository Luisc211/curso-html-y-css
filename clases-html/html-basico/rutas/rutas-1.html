Rutas en HTML

Una ruta en HTML (o path) es la dirección que nos indica dónde está ubicado un archivo, ya sea dentro de nuestro proyecto o en Internet.

Algunas etiquetas que usan rutas

  • <a href="ruta.html">Enlace</a>
  • <img src="ruta.jpg" alt="Imagen" title="Imagen">
  • <link rel="stylesheet" href="ruta.css">
  • <script src="ruta.js"></script>

Tipos de rutas

Rutas relativas

Indican la ubicación del archivo que necesitamos en relación con la página actual. Son las más comunes cuando trabajamos con archivos dentro de nuestro proyecto.

Mismo nivel

Para acceder a archivos que se encuentran en la misma jerarquía, basta con escribir el nombre del archivo al que queremos acceder.

<img src="amazonas.jpg" alt="Río Amazonas" title="Río Amazonas">

Una carpeta adelante

Para acceder a un archivo que se encuentra dentro de una carpeta, lo hacemos así: carpeta-contenedora/archivo. Es decir, escribimos el nombre de la carpeta que contiene el archivo, seguido de una barra / y luego el nombre del archivo.

<a href="carpeta-contenedora/archivo-contenido">Enlace</a>

Enlace

Más de una carpeta adelante

Para acceder a archivos que están dentro de varias carpetas, usamos una barra / por cada carpeta: carpeta-1/carpeta-2/nombre-del-archivo.

<a href="carpeta-1/carpeta-2/nombre-del-archivo">Enlace</a>

Enlace

Carpetas atrás

Para acceder a archivos que se encuentran en carpetas anteriores (niveles superiores), usamos ../ por cada carpeta que necesitemos retroceder. Ejemplos:

  • Una carpeta atrás:../
  • Dos carpetas atrás: ../../
  • Y así sucesivamente...

<a href="../../">Enlace</a>

Enlace

Rutas absolutas

Indican la dirección completa (incluyendo el protocolo y el dominio). Se utilizan para acceder a archivos que están fuera de nuestro proyecto, como imágenes o recursos en Internet. Por ejemplo, usando http o https:

<img src="https://www.ejemplo.com/imagen.jpg" alt="Imagen externa">
