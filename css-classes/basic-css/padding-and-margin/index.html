Padding and margin (Relleno y margen)

Padding

El paddin es ese espacio entre el contenido y el borde de esa caja, aumenta el tamaño que hay al rededor del contenido pero dentro de los bordes, el contenido siempre medira lo mismo, pero la distancia entre el contenido y el borde aumenta

Margin

El margin define la separación de una caja con otras cajas, hace que la caja ocupe más espacio, es decir si la caja mide 100px y le agregamos un margen de 10px a cada lado, ya medira 120px 10 de cada lado y 100px
