Fondos

Background repeat

  • Repeat: Se repite horizontal y verticalmente (valor por defecto)
  • no-repeat: No se repite en ninguna dirección
  • Repeat-x: Solo se repite horizontalmente
  • Repeat-y: Solo se repite verticalmente
  • Round: Ajusta el tamaño para repetir sin cortar
  • Space: Deja espacio entre repeticiones

Background-size

  • Auto: Tamaño original
  • Cover: Cubre todo el contenedor (recorta si es necesario)
  • Repeat: Se ajusta para no recortarse, pero puede dejar espacios
  • 100% 100%: Ancho y alto personalizados
