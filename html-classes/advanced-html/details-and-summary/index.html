Details y summary

Details y summary son etiquetas HTML que nos permite crear secciones desplegables sin necesidad de JavaScript, es decir podemos ocultar o mostrar información con un clic

Details

Es el contenedor que guarda el contenido que se puede abrir o cerrar

Summary

Es el titulo o encabezado que el usuario ve y sobre el que hace clic para abrir/cerrar la información.

¿Como funcionan?

Cuando el usuario hace clic en el summary, el contenido dentro de details se muestra

  • Si el usuario vuelve a hacer clic, el contenido se oculta
  • Por defecto, details esta cerrado
  • Si queremos que aparezca abierto, podemos usar el atributo open

Usos comunes

  • Preguntas frecuentes (FAQ)
  • Mostrar detalles opcionales en formularios
  • Información que no deseamos mostrar de inmediato
  • Listas largas que debemos ocultar
  • Explicaciones adicionales, pasos, notas, etc.

Ventajas

  • No necesitamos JavaScript
  • Es accesible, (lectores de pantalla lo interpretan correctamente)
  • Es fácil de usar
  • Funcionan en la mayoria de navegadores modernos
