Listas

En HTML tenemos dos tipos de listas:
<ol> (ordered list o lista ordenada) y <ul> (unordered list o lista desordenada).

El uso de una u otra dependerá de lo que necesitemos listar.
Si necesitamos crear una lista de pasos o un “top”, usamos <ol> porque el orden importa.
Si necesitamos crear una barra de navegación, usamos <ul> ya que el orden no importa.

Ambas tienen en su interior elementos llamados <li>, que representan los elementos a listar.

Ejemplos

Lista ordenada

  1. Elemento 01
  2. Elemento 02
  3. Elemento 03
  4. Elemento 04

Lista desordenada

  • Elemento 01
  • Elemento 02
  • Elemento 03
  • Elemento 04
