Formularios

Un formulario se crea con la etiqueta <form> y sirve para enviar datos desde el navegador a un servidor, por ejemplo, para iniciar sesión, registrarse o enviar un mensaje. Generalmente, utiliza el método post. Dentro de la etiqueta <form> se colocan las etiquetas <input>, que son campos en los cuales el usuario puede escribir texto, un correo electrónico o una contraseña. También permiten, por ejemplo, ingresar su fecha de nacimiento, elegir un color o subir un archivo.
