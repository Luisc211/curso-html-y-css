Estilos en línea

Los estilos en línea permiten aplicar formato directamente a un elemento HTML mediante el atributo style. Sin embargo, presentan algunas limitaciones. La principal es que constituyen una mala práctica, ya que mezclan el código HTML con el CSS, lo cual dificulta la lectura y el mantenimiento del código. Cada lenguaje debe mantenerse en su sección correspondiente para asegurar una estructura clara y ordenada.
